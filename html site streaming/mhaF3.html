MHA: world hereos mission

Nom Japonais : Boku no hero academia : The movie 3

Nom Anglais : My hero academia : The movie 3

Start date: 2016- en cours

Genre(s) Action, Sci-Fi, super pouvoir

SYNOPSIS:

Dans ce film, une mystérieuse organisation dédiée à la destruction des personnes dotées d’Alters a lancé une menace et posé des bombes dans le monde entier. Les héros professionnels et les stagiaires se démènent pour trouver les bombes. Deku, Bakugo et Todoroki rencontrent alors Rody, un garçon vivant dans une caravane dans le quartier d’Oseon, et finissent par travailler avec lui.
